# 📚 BookCircle - Connect. Share. Read.

## 📌 Overview
**BookCircle** is a modern book sharing platform that connects book lovers worldwide. Whether you have digital PDFs or physical books, BookCircle makes it easy to share knowledge, discover new reads, and build meaningful connections with fellow readers in your community and beyond.

---

## 🎯 Key Features

✅ **Smart Book Sharing**: Share both digital PDFs and physical books with an intelligent request system  
✅ **Community Driven**: Connect with book lovers in your area and around the world  
✅ **Built-in Messaging**: Coordinate exchanges with secure messaging between users  
✅ **Advanced Search & Filters**: Find books by title, author, genre, language, format, and distance  
✅ **Trust & Reviews**: Rate and review exchanges to build a trusted community  
✅ **User Dashboards**: Manage your books, requests, and exchanges in one place  
✅ **Gamification**: Earn badges and level up as you share more books  
✅ **Responsive Design**: Fully optimized for desktops, tablets, and mobile devices  
✅ **Guest Browsing**: Explore books without signing up to see the platform's value  
✅ **Real-time Notifications**: Stay updated on requests, messages, and community activity  

---

## 🛠️ Technologies Used

**Frontend:** Next.js 14, React 18, TypeScript  
**UI Framework:** Tailwind CSS, shadcn/ui components  
**State Management:** Zustand with persistence  
**Icons:** Lucide React  
**Notifications:** Sonner (toast notifications)  
**Authentication:** Custom auth system (ready for Firebase integration)  
**Database:** Mock data (ready for Firebase/Supabase integration)

## 🚀 Installation Guide

### 📋 Prerequisites
- **Node.js** (v18 or higher)  
- **npm** or **yarn** package manager  

### 📥 Steps to Set Up

1️⃣ **Clone the Repository:**  
\`\`\`bash
git clone https://github.com/your-username/bookcircle-platform.git
cd bookcircle-platform
\`\`\`

2️⃣ **Install Dependencies:**  
\`\`\`bash
npm install
# or
yarn install
\`\`\`

3️⃣ **Start Development Server:**  
\`\`\`bash
npm run dev
# or
yarn dev
\`\`\`

4️⃣ **Access the Application:**  
Open your browser and go to:  
🌍 \`http://localhost:3000\`

## 🎮 Usage Guide

- Sign up or sign in to get started
- Browse books without creating an account

- Search and filter books by various criteria

- Manage your shared books
- View sent and received requests
- Access your chat conversations
- Track your community stats and achievements
